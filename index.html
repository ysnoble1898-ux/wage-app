근무수당 통합관리

인센티브·수당 · 연장/휴일근무 · 월별 정산

대시보드

데이터 현황과 빠른 시작 안내

시작하기

  1. [직원·수당단가] 탭에서 직원을 등록하고, 직원별 인센티브 단가와 시급을 설정합니다. 사람마다 다른 단가 설정이 가능합니다.
  2. [03 인센티브·수당] 탭에서 월별로 직원별 방문 건수와 특별수당을 입력하면 자동 계산됩니다.
  3. [연장·휴일근무] 탭에서 일자별 연장/야간/휴일 근무를 기록합니다. 시급 × 시간 × 가산율로 자동 계산.
  4. [월별 정산] 탭에서 직원별 월 합계 정산서를 확인·인쇄할 수 있습니다.
  5. 작업 후 우상단 [데이터 내보내기]로 JSON 백업을 저장하세요. [불러오기]로 언제든 복원할 수 있습니다.
💡 모든 데이터는 브라우저 메모리에 보관됩니다. 페이지를 새로고침하기 전에 반드시 [데이터 내보내기]로 JSON 파일로 저장해주세요.

📋 처음 사용 시 — 5단계 설정

1

자동저장 활성화 필수 권장

상단 우측 ⚠️ 자동저장 미설정 칩을 클릭 → 📄 새 데이터 파일 만들기 선택. 파일을 지정하면 이후 모든 변경사항이 자동 저장됩니다.

⚠️ Chrome / Edge / Opera 전용. Safari·Firefox·모바일 미지원.
2

직원 등록 + 수당 단가 설정

[02 직원·수당단가] → ＋ 직원 추가

  • 기본 정보 (이름·직급·부서·입사일)
  • 기본급 (4대보험 산정 기준 월정액)
  • 방문 건당 단가: 일반/야간/주말/휴일 가산
  • 추가 건당 항목: 추가인센·응급콜·개인집·백신/검진·엔진오일 단가
    📌 추가인센 적용 시작 건수 설정 시 임계값 초과분 자동 계산
  • 월정액 수당: 차량유지비·위험수당·특수업무수당
  • 공제 항목: 부양가족·자녀 수 입력 후 ⚡ 자동계산 적용
    📌 국민연금 수령 연령 도달자는 "🔕 국민연금 면제" 체크
  • 통상시급 (월급 ÷ 209시간) 및 연장근무 가산율
  • 미사용 연차일수 (자동으로 연차수당 계산)
3

인센티브·수당 입력

[03 인센티브·수당] → ＋ 월별 기록 추가

  • 나이트전담 인센티브: 야간 전담자 정액 인센티브 (해당자만 입력)
  • 기타 항목: ＋ 항목 추가 버튼으로 내용과 금액을 자유롭게 기재
  • 방문 건수: 일반·야간·주말·휴일별 방문 건수 입력
  • 임계값 설정 직원은 초과분이 추가인센 칸에 자동 채워짐
  • 우측 미리보기 패널에서 실시간 합계 확인 가능
4

연장·휴일근무 입력 (필요시)

[04 연장·휴일근무] 탭에서 일자별로 입력합니다. 시작/종료 시각 입력 시 시간 자동 계산, 유형(연장/야간/휴일 등)에 따라 가산율 자동 적용.
🖨 인쇄/PDF 버튼으로 시급·금액 제외한 근무 사실만 출력 가능.

5

월별 정산 · 급여명세서 출력

[05 월별 정산] 탭 → 정산 년월 선택

  • 🧮 공제액 계산기에 총 급여 입력 → 4대보험·소득세 즉시 계산 → "이 직원에 적용" 버튼으로 한번에 반영
  • 공제 항목 숫자를 직접 클릭하면 바로 수정 가능 (합계·실수령액 자동 재계산)
  • 🖨 급여명세서 인쇄/PDF → 인쇄 대화상자에서 "PDF로 저장" 선택
  • 결재칸 직책명(센터장/총무/원장)은 클릭 후 자유롭게 수정 가능

💡 핵심 기능 안내

📊 소득세는 매월 총 지급액 기준으로 자동 재계산됩니다

[중요] 인센티브·수당과 연장근무 수당도 근로소득에 포함되므로, 매월 변동되는 총 지급액 기준으로 소득세를 재계산합니다.

  • 4대보험: 직원 단가에 저장된 값 그대로 (실무상 연 1회 신고 후 1년 고정)
  • 소득세·지방소득세: 매월 총 지급액 기준 자동 재계산
  • 수동 입력: 직원 단가 모달 → 소득세 항목 → "직접 입력값 사용" 체크
⚠️ 자체 계산값은 국세청 간이세액표와 차이날 수 있습니다. 정확한 값은 국세청 손택스 또는 소득세 계산기에서 조회 후 직접 입력하세요.
🧮 공제액 계산기 사용법 (월별 정산 탭)
  1. 월 총 급여액 입력 (기본급 + 인센티브 + 연장수당 합계)
  2. 부양가족 수·자녀 수·연도 선택 / 국민연금 면제 해당자는 체크
  3. 결과 확인 후 외부 링크(4대보험 계산기 / 손택스)에서 검증
  4. "이 직원 공제항목에 적용" 버튼으로 직원 단가에 한번에 저장
💡 4대보험은 보통 기본급 기준, 소득세는 총 지급액 기준으로 산출합니다.
✏️ 급여명세서 공제 항목 직접 수정
  • 공제 항목 숫자 클릭 → 바로 수정 → 공제합계·실수령액 즉시 재계산
  • 소득세·지방소득세 수정 시 자동으로 수동 입력 모드 전환
  • 수정값은 직원 공제 기본값으로 저장 / 인쇄 시 입력란 숨김
🔕 국민연금 면제 설정

[02 직원·수당단가] → 직원 수정 → 공제 항목 → 국민연금 항목 하단 체크박스

  • ⚡ 자동계산 적용 시 국민연금 자동 0원 처리
  • 정산서에 "국민연금 🔕" 표시 및 편집 비활성화
  • 해당 대상: 60세 이상, 노령연금 수급자, 공무원·군인·교직원연금 가입자 등
🌙 나이트전담 인센티브 + 기타 항목 입력

[03 인센티브·수당] → 월별 기록 추가/수정 → 상단 두 섹션

  • 나이트전담 인센티브: 야간 전담 근무자에게 월 정액으로 지급되는 인센티브 금액 직접 입력
  • 기타 항목: ＋ 항목 추가 버튼으로 행을 무제한 추가 — 내용(텍스트)+금액 자유 기재
    예) "4월 특별 업무비", "교육 참여 수당", "명절 선물비" 등
  • 급여명세서 지급 항목에 세부 항목별로 자동 반영
🖨 연장·휴일근무 기록 인쇄/PDF

[04 연장·휴일근무] 탭 → 년월·직원 필터 설정 → 🖨 인쇄/PDF 버튼

  • 출력: 일자·요일·직원·유형·시작·종료·시간·사유/승인
  • 제외: 시급·가산율·금액·수정/삭제 (공식 제출용 깔끔 출력)
  • 특정 직원만 출력 → "직원" 필터에서 해당 직원 선택 후 인쇄
💾 데이터 저장 방식
  • 자동저장 활성화 시 (권장): 입력 즉시 0.6초 후 JSON 파일에 자동 저장
  • 수동 백업: 우상단 📥 백업 다운로드 (월말마다 권장)
  • 다른 PC 이동: JSON 파일을 USB/클라우드로 복사 → 📤 백업 복원으로 불러오기
⏱ 추가인센 임계값 활용 팁
  1. 직원 단가 → 추가인센 적용 시작 건수에 임계값 입력 (예: 120)
  2. 인센티브·수당 입력 시, 총 방문이 임계값 초과분을 자동으로 추가인센 명수에 채워줌
  3. 필요시 수동 수정 가능 (수동 입력 우선)
🌴 연차수당 자동 계산
직원 단가 → 연차 잔여일수 입력 → 정산 시 통상시급 × 8시간 × 연차일수로 연차수당 자동 계산·명세서 반영
👥 결재칸 직책 변경
급여명세서 하단의 센터장 / 총무 / 원장 클릭 → 직접 수정. 변경 시 모든 직원 명세서에 동시 반영 (자동 저장).

🆘 문제 해결

증상해결 방법
자동저장이 안 됨Chrome / Edge 브라우저인지 확인. 시크릿 모드는 매번 재설정 필요. 자동저장 칩이 🔒면 클릭하여 권한 재허용.
인쇄 시 색이 빠짐인쇄 대화상자 → 설정 더보기 → "배경 그래픽" 체크
방문 건수 입력 후 인센티브가 0원직원·수당단가 → 해당 직원 수정 → 건당 기본 인센티브 단가 입력 여부 확인
4대보험이 너무 적게 나옴직원 단가 → 공제 항목 → 부양가족 수 확인 → ⚡ 자동계산 적용
소득세가 실제와 다름손택스·소득세 계산기에서 총급여 기준 조회 후, 공제 계산기 "직원에 적용" 또는 명세서에서 직접 수정
실수로 데이터를 지웠음최근 백업 JSON → 📤 백업 복원으로 불러오기 (자동저장 모드면 그 파일이 항상 최신본)
여러 PC에서 사용USB 또는 Google Drive에 JSON 보관 → 사용 PC에서 📂 기존 파일 불러오기로 자동저장 연결
